측면 패널에서 파일을 보고 열기
텍스트 편집기에서 파일을 전환하는 일반적인 방법은 텍스트 편집기 창의 맨 위에 있는 탭을 사용하는 것입니다. 그러나 많은 수의 열린 파일을 작업하는 경우와 같이 필요에 따라 측면 패널을 활용할 수 있습니다. 측면 패널을 사용하도록 설정하려면 View ▸ Side Panel(보기 ▸ 측면 패널)을 선택하거나 F9 키를 누릅니다. F9를 눌러 측면 패널을 언제든 열거나 닫을 수 있습니다.


측면 패널에서 파일 열기 
측면 패널에서 파일을 열려면 먼저 View ▸ Side Panel(보기 ▸ 측면 패널)을 선택하여 측면 패널을 연 다음 패널 상단의 드롭다운에서 File Browser(파일 검색기)를 선택합니다.

그러면 측면 패널의 File Browser(파일 검색기) 모드가 활성화됩니다. 그런 다음 패널 상단에 있는 탐색 버튼을 사용하여 원하는 파일을 찾아 열 수 있습니다.


열려 있는 파일 간 전환 
파일이 여러 개 열려 있으면 측면 패널을 사용하여 열려 있는 파일 사이를 전환할 수 있습니다. 열려 있는 파일을 나열하려면 패널 상단의 드롭다운에서 문서를 선택합니다.

측면 패널에서 파일 이름을 클릭하면 해당 파일이 편집용으로 열립니다.
