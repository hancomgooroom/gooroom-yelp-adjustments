바로 가기 키
작업을 더 빨리 수행하려면 바로 바로 가기 키를 사용합니다. 다음 표에서는 텍스트 편집기의 모든 바로 가기 키를 나열합니다.


- 탭 관련 단축

동작바로 가기 키
왼쪽의 다음 탭으로 전환Ctrl + Alt + Page Up
오른쪽의 다음 탭으로 전환Ctrl + Alt + Page Down
닫기 탭Ctrl + W
모든 탭 저장Ctrl + Shift + L
모든 탭 닫기Ctrl + Shift + W
가장 최근에 닫힌 탭 다시 열기Ctrl + Shift + T
n번째 탭으로 이동Alt + n
새 탭 그룹Ctrl + Alt + N
이전 탭 그룹Shift + Ctrl + Alt + Page Up
다음 탭 그룹Shift + Ctrl + Alt + Page Down


- 파일 작업을 위한 바로 가기 키

동작바로 가기 키
새 창에 새 문서 만들기Ctrl + N
새 탭에서 새 문서 만들기Ctrl + T
문서 열기Ctrl + O
Quick Open 창 열기Alt + O
현재 문서 저장Ctrl + S
새 파일 이름으로 현재 문서 저장Ctrl + Shift + S
현재 문서 인쇄Ctrl + P
인쇄 미리 보기Ctrl + Shift + P
현재 문서 닫기Ctrl + W
종료Ctrl + Q


- 파일 편집을 위한 바로 가기 키

동작바로 가기 키
현재 줄의 시작 부분으로 이동Home
현재 줄의 끝으로 이동End
문서 시작 부분으로 이동Ctrl + Home
문서의 끝으로 이동Ctrl + End
선택한 단어를 한 단어 오른쪽으로 이동Alt + →
선택한 단어를 한 단어 왼쪽으로 이동Alt + ←
마지막 작업 실행 취소Ctrl + Z
마지막으로 실행 취소된 작업 다시 실행Ctrl + Shift + Z
선택한 텍스트 또는 영역을 잘라내어 클립보드에 놓기Ctrl + X
선택한 텍스트 또는 영역을 클립보드에 복사Ctrl + C
클립보드의 내용 붙여넣기Ctrl + V
파일의 모든 텍스트 선택Ctrl + A
현재 줄 삭제Ctrl + D
선택한 줄을 한 줄 위로 이동Alt + ↑
선택한 줄을 한 줄 아래로 이동Alt + ↓
선택한 텍스트를 대문자로 변환Ctrl + U
선택한 텍스트를 소문자로 변환Ctrl + L
선택한 텍스트의 대/소문자 전환Ctrl + ~


- 창을 표시하고 숨기기 위한 바로 가기 키

파일 메뉴 열기F10

검색 및 바꾸기Ctrl + H

Goto lineCtrl + I
