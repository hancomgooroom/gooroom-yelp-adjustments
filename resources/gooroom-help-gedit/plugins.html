플러그인 구성 및 사용
플러그인을 사용하여 텍스트 편집기로 더 많은 작업을 수행할 수 있습니다. 기본적으로 여러 플러그인이 설치되지만 필요에 따라 추가 플러그인을 사용할 수도 있습니다.
대부분의 Linux 배포판에서는 플러그인 그룹을 gedit-plugins 패키지로 사용할 수 있습니다. gedit-plugins 패키지를 설치하면 추가 플러그인을 텍스트 편집기에 사용할 수 있습니다.
         
sudo apt-get install gedit-plugins


기본 플러그인
대소문자 바꾸기

텍스트를 선택하여 대소문자를 바꿉니다.
 
Python 콘솔

아래쪽 창에 대화형 Python 콘솔을 추가합니다.
 
문서 통계

현재 문서에 대한 다양한 통계를 보여줍니다.
 
빠르게 열기

파일 또는 문서를 빠르게 엽니다.
 
외부 도구

현재 파일에서 스크립트를 실행하고 다른 애플리케이션과 상호 작용합니다.
 
관용구

관용구를 사용하여 자주 사용하는 텍스트를 빠르게 삽입합니다.
 
파일 브라우저 패널

가장자리 창을 사용하여 파일을 찾아 엽니다.
 
정렬

텍스트 행을 알파벳 순서로 정렬합니다.
 
날짜/시간 삽입

커서 위치에 현재 날짜/시간을 삽입합니다.
 
맞춤법 검사

파일에 철자 오류가 있는지 확인합니다.
 
Modelines

텍스트 편집기에서 지원하는 Emacs, Kate 및 Vim 스타일의 Modelines를 사용합니다.
