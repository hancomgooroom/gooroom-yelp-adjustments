네트워크 사용
네트워크 사용 여부를 사용자가 설정할 수 있습니다. 기존에 연결된 유선 네트워크 또는 와이파이가 없는 경우 네트워크가 비활성화됩니다. 새로 유선 네트워크 또는 와이파이를 연결하는 경우 자동으로 네트워크가 활성화됩니다. 유선 네트워크 또는 와이파이가 연결된 상태에서 네트워크 비활성화가 필요한 경우 네트워크 사용 체크 박스를 해제하여 네트워크를 비활성화 할 수 있습니다.

다음 단계를 수행하여 네트워크 사용 여부를 설정합니다.


1. 우측 하단의 [네트워크] ( (유선 네트워크), (와이파이), (네트워크 미연결)) 아이콘을 클릭하여 네트워크 메뉴를 엽니다. [네트워크 사용] 체크 박스를 해제하여 네트워크 연결을 비활성화 할 수 있습니다.


          


2. [네트워크 사용] 체크 박스를 체크하여 네트워크 연결을 활성화 할 수 있습니다. 유선 네트워크 연결이 된 경우 [유선 네트워크 연결]  아이콘을, 유선 네트워크 연결 없이 와이파이 연결만 된 경우 [와이파이 연결]  아이콘을 확인할 수 있습니다.


1) 유선 네트워크

          


2) 와이파이
