연결 종류 선택
네트워크 연결 생성 시 네트워크 연결 종류를 선택할 수 있습니다. 구름 OS는 와이파이, 이더넷, 블루투스 등의 네트워크 연결 종류를 지원합니다.

다음 단계를 수행하여 네트워크 연결 종류를 선택할 수 있습니다.


1. 우측 하단의 [네트워크] ((유선 네트워크), (와이파이), (네트워크 미연결)) 아이콘을 클릭하여 네트워크 메뉴를 엽니다. [연결 편집] 메뉴를 클릭합니다.


          



2. 나타난 네트워크 연결 창에서 현재 연결된 네트워크 종류를 확인할 수 있습니다. 새로운 네트워크 종류는 좌측 하단 [추가]  아이콘을 클릭하여 연결할 수 있습니다. 


          



3. 나타난 연결 종류 선택 창의 드롭다운 목록에서 원하는 연결 종류를 선택 후 [만들기] 버튼을 클릭하여 새 네트워크를 생성할 수 있습니다.
