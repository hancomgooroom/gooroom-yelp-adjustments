설치
소프트웨어 센터에서 특정 앱을 클릭하여 앱 상세화면 페이지로 이동 후 우측 상단의 [설치]를 클릭하여 해당 소프트웨어를 설치할 수 있습니다.


          



앱 설치가 끝나면 [실행]을 클릭하여 실행할 수 있습니다.


          



설치된 앱은 좌측 하단 [시작]  아이콘을 클릭하여 확인하거나, 소프트웨어 앱 삭제에서 확인할 수 있습니다.
