글자 크게 보기 설정
화면에 표시된 글자가 작아 읽는 데 어려움이 있는 경우 글자 크기를 크게 설정할 수 있습니다.

다음 단계를 수행하여 글자 크기를 크게 변경합니다.


1. 좌측 하단의 [시작]  아이콘을 클릭하고 검색 창에 "설정"을 입력한 후 [설정] 애플리케이션을 실행합니다.


          

          


2. [설정] 애플리케이션 사이드 바에서 [접근성]을 클릭합니다.


          

          


3. [보기] 아래에서 [큰 글씨] 설정 단추를 오른쪽으로 밀어 켭니다.


          

대부분의 애플리케이션에서 Ctrl + + 키를 누르면 글자 크기가 커지고, Ctrl + - 키를 누르면 글자 크기를 작아집니다.
