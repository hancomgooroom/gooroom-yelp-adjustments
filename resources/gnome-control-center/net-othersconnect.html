다른 사용자가 인터넷에 연결할 수 없음
사용자가 네트워크 연결을 설정할 때 일반적으로 컴퓨터에 있는 다른 모든 사용자들도 네트워크를 사용할 수 있습니다. 네트워크 연결 정보가 공유되지 않은 경우 연결 설정을 확인해야 합니다.

다음 단계를 수행하여 모든 사용자의 인터넷 연결 상태를 확인합니다.


1. 좌측 하단의 [시작]  아이콘을 클릭하고 검색 창에 "설정"을 입력한 후 [설정] 애플리케이션을 실행합니다.


          

          


2. [설정] 애플리케이션 사이드 바에서 [와이파이]를 클릭합니다.

3. 연결된 네트워크 옆 [설정] 아이콘을 누르고 [자세히 보기] 탭에서 [다른 사용자가 사용할 수 있게 허용]을 선택합니다. 

4. [적용]을 눌러 변경 사항을 저장합니다.
