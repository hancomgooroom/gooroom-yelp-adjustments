추가 모니터 연결
2개 이상의 모니터를 사용하려면 컴퓨터와 추가 모니터를 연결해야 합니다. 

추가 모니터가 시스템에서 즉시 인식되지 않거나 이와 관련된 설정을 변경하려면 다음을 수행합니다.


1. 좌측 하단의 [시작]  아이콘을 클릭하고 검색 창에 "설정"을 입력한 후 [설정] 애플리케이션을 실행합니다.


          

          


2. [설정] 애플리케이션 사이드 바에서 [장치]를 클릭한 후 [디스플레이]를 클릭합니다.


          

          


3. 디스플레이가 정렬된 다이어그램에서 디스플레이를 드래그하여 사용자가 원하는 상대적인 위치에 놓습니다. (다이어그램에서 나타나는 숫자와 디스플레이 좌측 상단에 나타나는 숫자를 비교하여 모니터를 구분할 수 있습니다.)

4. 기본 디스플레이로 사용할 모니터에서 기본 디스플레이를 클릭합니다. (메인 디스플레이에는 메뉴 아이콘을 포함한 상태 표시줄이 표시됩니다.)

5-1. 각 모니터에 대한 방향, 해상도, 주사율 등을 설정합니다.

5-2. 변경된 내용을 적용하지 않으려면 [되돌리기]를 선택하거나 15초 기다린 후 3-5번 과정을 반복합니다.
디스플레이 모드
2개 이상의 모니터를 컴퓨터에 연결하면 다음과 같은 디스플레이 모드를 사용할 수 있습니다.


디스플레이 연결 - 화면의 가장자리가 연결되어 한 디스플레이에서 다른 디스플레이로 이동이 가능합니다.

거울 - 동일한 컨텐츠가 두 디스플레이에 나타납니다. 거울 모드인 경우 방향 및 해상도가 동일합니다.

단일 디스플레이 - 1개의 디스플레이만 사용합니다. 외부 모니터에 연결된 노트북이 단일 디스플레이 모드에 해당됩니다.
3개 이상의 모니터 연결
3개 이상의 모니터를 연결하는 경우 디스플레이 모드 중 [디스플레이 연결]만 사용할 수 있습니다.

다음 단계를 수행하여 3개 이상의 모니터를 설정합니다. 


1. 각 디스플레이에서 버튼을 눌러 모니터를 구별합니다.

2. 구별된 디스플레이를 드래그하여 상대적인 위치를 결정합니다.
